Docker on window初体验

1.安装docker on window

1、Docker Compose for Window 不知道什么用，勾选
2、VirtualBox 虚拟机(可以试试不安装这个，如果Bash操作的话)
3、Kitematic for Windows(Alpha) 使用图形界面来使用 docker(可以试试不安装这个，如果Bash操作的话)
4、Git for Windows 一个版本控制 + bash 命令终端，如果没有安装建议勾选

2、启动Docker


启动docker toolbox的时候，会出现找不到而去下载boot2docker.iso的情况，断开网络，拷贝安装目录下的boot2docker.iso到目的路径即可。

3、遇到问题

直接使用docker run 运行ubuntu的镜像时会出现”the input device is not a TTY. If you are using mintty, try prefixing the command with ‘winpty’”的错误，前面加上winpty即可。

4.完毕
